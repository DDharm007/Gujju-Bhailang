General

Gujju-BhaiLang is a toy programming language inspired by Gujarati culture. Every program must start with start: and end with end: in simple mode, or sharu: and samaapt: in abusive mode. Anything outside of it will be ignored.

This will be ignored
start:
  // Write code here
end:
This too
This will be ignored
sharu:
  // Write code here
samaapt:
This too

Variables

Variables can be declared using simple assignment in simple mode, or loda ayia nakh in abusive mode.

start:
  umar = 25;
  naam = "Bhai";
  sachu = true;
  umar = umar + 1;
  naam = "New Name";
end:
sharu:
  loda ayia nakh umar = 25;
  loda ayia nakh naam = "Bhai";
  loda ayia nakh sachu = true;
  umar = umar + 1;
  naam = "New Name";
samaapt:

Types

Numbers and strings are like other languages. Null values can be denoted using nalla. sachu and khotu are the boolean values.

start:
  umar = 25;
  total = 10 + (15 * 20);
  naam = "Bhai";
  char = 'a';
  empty = nalla;
  flag = sachu;
  wrong = khotu;
end:
sharu:
  loda ayia nakh umar = 25;
  loda ayia nakh total = 10 + (15 * 20);
  loda ayia nakh naam = "Bhai";
  loda ayia nakh char = 'a';
  loda ayia nakh empty = nalla;
  loda ayia nakh flag = sachu;
  loda ayia nakh wrong = khotu;
samaapt:

Built-ins

Use print: in simple mode or lakh chodya: in abusive mode to print anything to console.

start:
  print: "Hello World";
  umar = 10;
  {
    naam = "Bhai";
    print: umar + naam;
  }
end:
sharu:
  lakh chodya: "Hello World";
  loda ayia nakh umar = 10;
  {
    loda ayia nakh naam = "Bhai";
    lakh chodya: umar + naam;
  }
samaapt:

Conditionals

Use jo for if statements, to for then, and nahi to for else.

start:
  umar = 20;
  jo (umar > 18) to {
    print: "Adult cho";
  } nahi to {
    print: "Balak cho";
  }
end:
sharu:
  loda ayia nakh umar = 20;
  jo (umar > 18) to {
    lakh chodya: "Mota cho";
  } nahi to {
    lakh chodya: "Chodya balak";
  }
samaapt:

Loops

Use repeat X vaar to execute code multiple times.

start:
  repeat 5 vaar {
    print: "Maja ma che";
  }
end:
sharu:
  repeat 5 vaar {
    lakh chodya: "Maja avi gai";
  }
samaapt:

Functions

Create reusable code blocks using function keyword.

start:
  function bolavo(naam) {
    print: "Kem cho " + naam;
  }
  bolavo("Bhai");
end:
sharu:
  function loda_bolavo(naam) {
    lakh chodya: "Kem cho loda " + naam;
  }
  loda_bolavo("Bhai");
samaapt:

Arrays

Store multiple values using list in simple mode or loda ayia nakh list in abusive mode.

start:
  list mitro = ["Ram", "Shyam", "Mohan"];
  print: mitro[0];
  print: "Total mitro: " + mitro.length;
end:
sharu:
  loda ayia nakh list lodo = ["Ram", "Shyam", "Mohan"];
  lakh chodya: lodo[0];
  lakh chodya: "Total lodo: " + lodo.length;
